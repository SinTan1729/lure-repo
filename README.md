## LURE Repository

This is a repository for my personal [LURE](https://github.com/SinTan1729/lure) (my-fork) build scripts. Of course, anyone else is welcome to use these.

These are written for RPM-based distros (I'm testing on AlmaLinux 9.1), but should work with most other distros as well.

## Installation

This is the default repo on my fork of LURE.

If using the official version, install this repo.

```
lure addrepo -n SinTan1729 -u https://github.com/SinTan1729/lure-repo
```
